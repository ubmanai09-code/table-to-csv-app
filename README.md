# Table to CSV App

This application converts table data into CSV format and provides options for users to customize their CSV output.

## Features
- Import table data from various sources.
- Export formatted CSV files.
- User-friendly interface.

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `npm start`.

## Usage
- Navigate to the application in your browser.
- Input your table data.
- Click on 'Export CSV' to download your file.
